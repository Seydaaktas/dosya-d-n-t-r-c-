# dosya-d-n-t-r-c-
Farklı dosya formatları arasında dönüşüm yapmayı sağlayan, Java tabanlı ve esnek mimarili bir masaüstü uygulamasıdır. Proje, her dosya biçimini bağımsız birer nesne olarak ele alarak spagetti kod yapısının önüne geçer ve yüksek genişletilebilirlik sunar. Tamamen modüler ve SOLID prensipleri gözetilerek inşa edilmiştir.
